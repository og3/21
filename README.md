# 「21」とは
- カードゲーム「21」を題材にしたCUIゲーム
# ゲームのルール
## ゲームの盤面について
- ゲームの盤面には以下の要素がある。
- 山札（山札となるカードは1～11の数字が書かれたカードが1セット11枚しかない。なので、場に出ているカードと同じカードを引けない。）
- 自分と相手のプレイヤーの手札（1枚を除く、すべてのプレイヤーが見える状態）
- 5本指を模したライフカウンター（マッチに敗北すると減っていく）
## 勝敗の条件
- 交互に数字が書かれたカードを引き、手持ちカードの合計がより「21」に近い方が勝利となる。
- 手持ち合計値21を超えると「バースト」となり、相手もバーストしていなければ負けとなる。
- お互いが同じ数字なら引き分け。
## ゲームの進行
- 1,ゲーム開始時にカードが2枚ずつ配られる。(1枚目のみ伏せられており、数字が分からない。) 
- 2,プレイヤーのターンとなり、カードを1枚引くかどうかを選択する。
- 3,相手プレイヤーのターンとなり、同様の選択を行う。
- 4,お互いにカードを引かない場合は勝敗判定を行う。
- 5-1,敗北したプレイヤーのライフカウンターを減らし、お互いにライフカウンターが0でなかったら、次のラウンドとして1からやり直す。
- 5-2,いずれかのプレイヤーのライフカウンターが0になったらゲームを終了し、勝利プレイヤーを宣言する。
